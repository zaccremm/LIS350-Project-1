This volvelle is a lunar clock from Peter Apianus' "Cosmographia" [7]. The owner would match up the top piece ("Piece 2" slider) to the current moon phase, then match the middle piece ("Piece 1" slider) to the current time. Doing so would give a model that you can use to determine the location of the sun, even at night [8].
